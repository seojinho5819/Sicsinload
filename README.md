# React-Project-sicsinload
React를 이용한 Naver map API기반 맛집 리스트


## 🖥️ 프로젝트 소개
데이터 최신화를 위해 맵 구현은 Open API, 데이터는 실제 MAP API기반으로 구현하였습니다.
<br>

## 🕰️ 개발 기간
* 22.04.26일 - 22.05.24일

### ⚙️ 개발 환경
- `React`
- `NodeJs v14.18.2`
- `npm v6.14.15`
- **IDE** : vscode
- **Framework** : react(^18.2.0)
- **storage** : local storage
